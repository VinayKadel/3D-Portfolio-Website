# Vinay Kumar Soni — 3D Portfolio Website

Welcome to the source code for my interactive 3D portfolio! This is a modern, one-page web application built to showcase my skills in Full Stack Development, the MERN stack, Next.js, and AI Integrations.

## 🚀 About This Project

This portfolio utilizes React Three Fiber and GSAP to create a rich, immersive 3D experience. It features interactive 3D elements, smooth scroll-driven animations, custom cursor interactions, and a sleek, fully responsive design. 

## 🛠️ Tech Stack

- **Frontend:** React, Next.js, TypeScript, Tailwind CSS
- **3D & Animation:** Three.js, `@react-three/fiber`, `@react-three/drei`, GSAP
- **Backend / Database:** Node.js, Express, MongoDB, Prisma
- **Tools & Deployment:** Vite, Git, Vercel

## 💻 Getting Started

### Prerequisites

- Node.js 18+ (recommended)
- npm 9+ (or compatible)

### Installation

1. **Clone the repository:**

   ```bash
   git clone https://github.com/VinayKadel/3D-Portfolio-Website.git
   cd 3D-Portfolio-Website
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Start the local development server:**

   ```bash
   npm run dev
   ```

4. **View in browser:**
   Open the URL shown in your terminal (typically `http://localhost:5173`).

## 📜 Available Scripts

- `npm run dev` — Starts the Vite development server.
- `npm run build` — Type-checks and builds a production-ready bundle.
- `npm run preview` — Serves the production build locally for verification.
- `npm run lint` — Runs ESLint checks across the project.
